Converters
==========

.. module:: smartconfig.converters

The :mod:`smartconfig.converters` module provides built-in converter functions
that transform raw configuration values into Python objects of the appropriate type.
See :doc:`default_converters` for usage examples and details on how converters
are applied during resolution.
